# NestJS 배우기

## Flow (↓)

- main.ts
- module
- controller : url매핑, request 받기
- service : 함수실행(DB 동작)

## Keyword

- decorator
  - @Get()
  - @Post()
  - @Patch()
  - @Delete()
  - @Put()

  - @Param()
  - @Query()
  - @Body()

## Single-Responsibility-Principle

- 하나의 `module`, `class`, `function` 은 반드시 하나의 기능만 하도록 한다.
